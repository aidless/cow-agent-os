## ❌ 我之前死磕的"v1.0.x 模板陷阱"已全部绕过

这是今晚的"教训标本",**对将来仍有价值**(作为反面教材):

1. ❌ **bat for 循环变量必须 `%%f`**(不是 `%f`、不是 `!f!`)
   - v1.0.1 用 `%f` → "此时不应有 f"
2. ❌ **bat 内出现 `FAIL`/`ERR`/`BAD` 单独成词** → cmd 当命令,假报错
   - v2.0.0 改用 `[X]/[OK]` 绕过
3. ❌ **`for /R` glob 会匹配无关目录同名文件** → 用 powershell 列真路径
   - v2.0.0 改用 `Get-ChildItem -Include *.json` + 排除 `.git`
4. ❌ **`node -e "...F:\test\..."` 在 cmd 里** → 反斜杠被 cmd 吞(变成空格、数字字符)
   - v2.0.0 改独立 .js 脚本 + `process.argv[1]` 拿路径,完全避开 cmd 引号处理
5. ❌ **`echo FAIL xxx` / `:runTest` 用了 `exit /b 1` 退出码** → 污染调用方
   - v2.0.0 改用 `RT_OK` 变量 + `goto :eof`
