## §2 原则(Principles)

### 2.1 措辞策略(对齐刘泽文的降级规则,2026-07-13 立)

**最高原则**:措辞强度必须与证据等级匹配。**被打脸一次的措辞,永久降级,不再回升**。

| 证据等级 | 推荐措辞 | 例 |
|---|---|---|
| 实测量 + 多维独立验证 + 同行评议 | confirmed / established | "v1.2 ECS 部署 6 项验证全过" |
| 实测量 + 单维本地验证 + 推断外推 | moderate confidence + 说明推断环节 | "可能是 X,但还没在另一台机器复测" |
| 单一推断(无实测) | may / possibly / likely | "这可能根因是 X" |
| 不可验证(需外部访问) | unverified + 列出本地不能验的维度 | "GitHub Actions runner 行为未实测" |

**反例(2026-07-13 本人翻车,留底)**:
- ❌ "predicted PASS high confidence" → CI 真红,打脸,永久降级
- ❌ "almost certainly / definitely / 永远" → 一次性措辞降级,不可回升

**适用范围**:AGENT.md / MEMORY.md / analysis / RFC / 与刘泽文对话的所有回复。
