## 🏆 立刻可激活的"宝石"(3 项)

### 1. `planning-with-files/` v3.4.0 ⭐⭐⭐⭐⭐

- **干什么**:**Manus 风格**的持久文件化规划 —— AI agent 在磁盘上维护 `task_plan.md` / `findings.md` / `progress.md`,作为"工作记忆"
- **为什么跟你的工作强相关**:**它的设计灵魂 = WORKFLOW.md 律令 0**(先口头扫盘再行动)的**产品化版本**
- **关键设计**:
  - **SHA-256 attestation** —— 防止 plan 文件被注入恶意指令
  - **`PreCompact` hook** —— context 压缩前自动提醒"先更新 progress.md"
  - **`session-catchup.py`** —— 新 session 自动 catch up 上一 session 的进度
  - **`task_plan.md` 在项目根,不在 skill 安装目录** —— 跟你"工作空间独立"哲学契合
- **何时激活**:**写 PAPER3-6 / 泰玄小站 v2.0 / 任何 5+ 步的任务之前**
- **激活方式**: `agentskills install <path>` 或加到 active skills 列表
- **可能风险**:v3 的 hook 链很长,跑通前需要看一遍 hooks 没冲突
