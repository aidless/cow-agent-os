# 微信小程序命理类目资质申请 · 全流程指南

> **适用项目**:泰玄小站(v1.9.12,v2.0 商业化前置)
> **法规框架**:微信小程序运营规范 + 工信部 ICP 备案 + 网信办生成式 AI 算法备案 + 广告法
> **核心难点**:**命理/占卜是微信受限类目,个人主体 100% 不能做**
> **预估时间**:5-7 个月(含反复驳回重审)

---

## 一句话核心

**5 步上线硬门槛,缺一不可**:

```
企业主体 → ICP 备案 → 算法备案号 → 微信类目 → 商户号
```

每一步都可能**驳回 2-3 次**,需要冗余 3-6 个月时间。**v2.0 上线预估:2026-09 至 2026-12 之间,取决于算法备案下证速度。**

## 第 1 章 · 5 步硬门槛清单(必读)

### 步骤 1 · 企业主体注册(1-2 周)

**为什么**:个人主体微信不接受命理/占卜/咨询类目。

**操作**:
- 工商注册"文化传播""网络科技""信息咨询"类公司
- 经营范围**避免** "命理、占卜、算命、宗教、心理咨询"
- 推荐命名:"XX 文化传媒有限公司" / "XX 科技有限公司"
- 注册资本:认缴 10-100 万即可(不需实缴)

**成本**:
- 自办:0 元 + 时间成本
- 代办:¥500-2000
- 财务记账(每月):¥200-500

**驳回原因 + 应对**:
- ❌ 经营范围含"命理" → 改名,经营范围只写"文化艺术交流、信息咨询"
- ❌ 主体名称"XX 命理公司" → 改名"XX 文化公司"
- ✅ 主体无违法记录(法人征信 + 公司无经营异常)

### 步骤 2 · ICP 备案(7-15 天)

**为什么**:小程序后端必须有 HTTPS 备案域名,微信不接受 HTTP 裸 IP。

**前置条件**:
- 已有企业主体(步骤 1 完成)
- 已有服务器 IP / 域名(`app.js:12 WEB_URL` 当前是 `http://116.62.69.83`,**必须替换**)
- 域名建议:`taixuan.cn` / `taixuan-app.com` / `fortune-web.com`(避免"命理""算命"词根)

**操作流程**:
1. 域名注册(阿里云/腾讯云):¥50-100/年
2. ICP 备案(阿里云/腾讯云代提交):0 元,但需 7-15 工作日
3. 备案服务内容选:**工具/生活服务/教育-其他**(避"占卜"作为服务名)

**成本**:¥50-100(域名) + 7-15 天时间

**驳回原因 + 应对**:
- ❌ 域名含"mingli / suanming / zhanbu" → 改名,见 §4 替代表达
- ❌ 服务内容写"占卜" → 改"文化工具 / 文化艺术"
- ❌ 网站未建好就备案 → 先建 1 个静态首页(含 ICP 备案号占位)

### 步骤 3 · 生成式 AI 算法备案号(2-6 个月 ⚠️ 最长)

**为什么**:2023.8 起,生成式 AI 服务必须备案。命理类解读 = AI 生成内容,**无备案号 = 违规**。

**前置条件**:
- 企业主体(步骤 1)
- ICP 备案(步骤 2)
- LLM 服务说明(模型选型 / 训练数据 / 安全机制 / 适用范围)
- 已部署的 AI 服务(可以先 demo,后端要能跑)

**操作流程**:
1. 登录 [网信办互联网信息服务算法备案系统](https://beian.cac.gov.cn)
2. 主体信息填报(企业营业执照)
3. 算法信息填报:
   - 算法名称:"泰玄传统文化解读算法"(不要"命理""算命")
   - 算法类型:生成式 AI / 对话式 AI
   - 应用场景:"文化体验类工具"
   - 模型来源:自研 / API 调用(对调用 GPT/Claude/DeepSeek 的要分别说明)
   - 训练数据来源:公开传统文化资料 + 自建模板库(注:不开源 6tail/lunar 等)
4. 安全机制:
   - 输出过滤(玄学内容关键词过滤)
   - 内容审核(人工复核)
   - 用户举报渠道
   - 数据保护说明
5. 提交 → 审核 → 公示(2-6 个月)

**成本**:0 元,但耗时最长。

**驳回原因 + 应对**:
- ❌ 应用场景写"命理预测" → 改"传统文化解读工具 / 文化体验服务"
- ❌ 没说明"不预测具体命运,只做传统文化视角下的解读" → 加免责声明到算法说明
- ❌ 没提供安全机制详情 → 加 4 条(过滤/审核/举报/保护)
- ❌ 模型来源不清(调用多个 API) → 列清楚每个 API + 用途

**核心技巧**:把"命理预测"包装为"传统文化视角下的解读工具"。**泰玄小站 v1.0 已经走这条路**(privacy 页主动声明 + CODEX 合规定位)。

### 步骤 4 · 微信小程序类目申请(1-2 个月,可能驳回 2-3 次)

**为什么**:微信类目审核是玄学类最大门槛。命理/占卜是受限类目。

**微信类目选项**(从低到高):
- ✅ "工具-效率" —— 包装为"命格解读工具",**通过率较高**
- ✅ "教育-其他" —— 包装为"传统文化学习工具"
- ⚠️ "生活服务-咨询" —— 需咨询类资质,成本高
- ❌ "命理/占卜/咨询" —— 严格受限,个人主体 100% 否,企业主体**审批**通过率 < 30%

**推荐路径**:**双层防御**
1. 主类目:**工具-效率 / 教育-其他**(包装)
2. 副类目:**生活服务-咨询**(如果资质够)

**操作流程**:
1. 微信公众平台 → 小程序 → 设置 → 类目管理
2. 添加类目 → 选择"工具-效率"(或教育-其他)
3. 上传资质:
   - 企业营业执照
   - ICP 备案号
   - 算法备案号(步骤 3)
   - 服务说明文档(参见 CODEX §5 红线避让清单)
4. 提交审核 → 7-15 工作日 → 通过/驳回
5. 驳回 → 修改服务说明 → 重新提交(平均 2-3 次)

**驳回原因 + 应对**:
- ❌ "小程序名称含'算命'" → 改名"泰玄小站"(已改,合规)
- ❌ "服务说明提'预测命运'" → 改"传统文化视角下的解读"
- ❌ "未提供算法备案号" → 提供步骤 3 的备案号
- ❌ "经营范围与小程序内容不符" → 公司经营范围加"文化艺术交流、信息咨询"

### 步骤 5 · 微信支付商户号(2-4 周)

**为什么**:v2.0 商业化需要支付能力(大师堂付费墙 + 会员订阅)。

**前置条件**:
- 企业主体(步骤 1)
- ICP 备案(步骤 2)
- 小程序类目已通过(步骤 4)
- 法人身份证 + 银行卡

**操作流程**:
1. 微信支付商户平台注册:pay.weixin.qq.com
2. 主体信息 + 法人验证
3. 结算账户绑定(对公账户,需去银行开通)
4. 提交审核 → 1-3 工作日 → 通过
5. 商户号绑定到小程序 appid
6. 在小程序后台开通"微信支付"功能

**成本**:0 元 + 1-3 天 + 对公账户管理费(¥100-300/年)

**驳回原因 + 应对**:
- ❌ 经营类目与小程序不符 → 确保步骤 4 类目已通过
- ❌ 法人身份证不一致 → 重传清晰扫描件
- ❌ 对公账户未开通 → 先去银行开通基本户

## 第 6 章 · 风险与回滚

### 场景 A:算法备案被驳回 2 次以上
- **响应**:调整算法说明措辞,不要"预测""占卜"等字眼;**重新提交**(平均驳回 2.2 次)
- **回滚**:备案拖到 6+ 月 → 推迟 v2.0 上线,先做 H5 + 公众号引流

### 场景 B:微信类目被拒
- **响应**:换类目("工具-效率" → "教育-其他");改服务说明
- **回滚**:换"双轨"策略,主推独立 App / H5,小程序作为引流

### 场景 C:商户号开通后被风控
- **响应**:确保小程序内容合规 + 不诱导分享 + 不虚假宣传
- **回滚**:关闭支付功能,做"内容免费 + 服务付费"模式(转私域)

### 场景 D:已上线被举报下架
- **响应**:紧急修改违规内容 → 申诉 → 7-15 天
- **预防**:`check-objective-language.js` + `audit-wxml-tone.js` **每个版本必跑**(动作 ② 验证流水线已自动化)
